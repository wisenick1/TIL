외부 조인
==
> 출처 : 김영한의 [김영한의 실전 데이터베이스 - 기본편](https://www.inflearn.com/course/김영한-실전-데이터베이스-기본편/dashboard)

교집합에 들어있지 않은 값을 보기 위해서는 left, right join 같은 외부 조인이 필요하다.

외부 조인
--
left join
- from을 기준으로 조인
- 기본적이로 left join이 사용된다.

right join
- join 절을 기준으로 조인

조인의 특징
--
- 자식이 부모를 조인할 때는 데이터가 늘어나지 않는다.

기준으로 삼는 행에 데이터가 하나씩 밖에 들어가지 않으니 데이터가 늘어나지 않음

- 그러나 부모가 자식을 조인할 때는 데이터가 하나의 행에 여러 개 들어갈 수 있으므로 중복될 수 있다.

:rocket: **이는 데이터가 중복된 상태에서 잘못된 집계 함수의 사용 등으로 이루어질 수 있으니 조심하자**

**또한, 기준으로 삼는 테이블, 증가할 것인가 미리 생각하고 쿼리 작성하는 습관 들이기**

cross join
--
데이터가 많을 경우 시간이 엄청나게 걸릴 수 있으니 이 점을 주의하자!
