상속관계 매핑
--
데이터베이스에서 상속 관계를 만들어주는 느낌

조인 젼략, 단일 테이블 전략, table per class 전략

strategy만 변경해주면 된다

되도록 조인 전략과 단일 테이블 전략 중에 상황에 맞춰 사용하자.
(단, 단일 테이블 전략은 DTYPE이 필수이다.)

@MappedSuperclass
--
상속 관계와 관련이 없다.

여러 테이블들에 공통되는 속성이 필요할 때 사용한다.

따로 테이블로 관리하는 것이 아니기 때문에 @Entity가 필요없다.
