배열
--
- 인덱스 사용하여 값에 바로 접근
- 새로운 값을 삽입, 특정 인덱스의 값 삭제가 어렵다 -> 다른 값들도 다 움직여야 한다
- 한번 선언하면 크기 늘리기, 줄이기 X

리스트(값 + 포인터)
--
- 인덱스 x -> Head 포인터부터 순서대로 접근/ 접근 속도 느리다(배열보다)
- 데이터 삽입, 삭제 속도 빠르다
- 크기 지정 X -> 크기 변하기 쉬운 데이터에 굿!

배열 VS 리스트
--
배열 - 간단, 크기 fix, 값에 접근이 많을 때

리스트 - 크기가 변함, 데이터 삽입과 삭제 많을 때
